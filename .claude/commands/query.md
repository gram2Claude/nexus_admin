Выполни **QUERY** по долгосрочной памяти проекта. Вопрос: $ARGUMENTS

0. Прочитай `.claude/memory_long/SCHEMA.md`.
1. Начни с **обоих** индексов: `.claude/memory_long/wiki/index.md` (публичный) и
   `.claude/memory_long/wiki/_private/index.md` (приватный дистиллят, если есть) — найди релевантные страницы.
2. Пройди по `[[ссылкам]]` вглубь, собирая контекст (при нехватке — `Grep`/`Glob` по `wiki/`).
3. Синтезируй ответ **только по прочитанным страницам**, с ссылкой `[[страница]]` на каждое утверждение.
   Цитаты из `_private/` помечай (приватный источник). Если ответа в вики нет — скажи честно и предложи
   добавить источник в `raw/` и сделать `/ingest`.
4. **Только чтение:** ничего не пиши в `.claude/memory_long/`.
