Выполни **LINT** долгосрочной памяти. Область: $ARGUMENTS

0. Прочитай `.claude/memory_long/SCHEMA.md` — это эталон проверки.
1. Прочитай `wiki/index.md`, `wiki/_private/index.md`, `log.md`; перечисли страницы (`Glob` по `wiki/` вкл. `_private/`).
2. Найди: **битые `[[ссылки]]`**; **сироты** (нет входящих ссылок); **недостающие связи** (unlinked mentions);
   **устаревшее** (по `updated`/датам/`git` mtime); **дубли**; **рассинхрон индексов**; нарушения формата/SCHEMA.
   НЕ ошибка: отсутствие `_private/` (склонировали без приватного); `source` на gitignored `raw/_large/`.
3. **ПОКАЖИ ОТЧЁТ** по категориям (файл + предлагаемая правка). Пока ничего не правь.
4. **ОСТАНОВИСЬ и жди**, какие правки применить.
5. После «да» примени правки по SCHEMA и допиши итог в соответствующий `log.md` (приватные правки → `_private/log.md`).

> Примечание: в движке есть `scripts/lint.py` (7 структурных + LLM-проверок) — после per-project порта (Ф2)
> эту команду можно свести к запуску `lint.py --mem-dir <...>`; пока работает как промпт-проверка по SCHEMA.
