Правила:
- На каждом уровне (эпоха/спринт): описание (если есть), даты, план-часы, статус
  (завершена / в работе / в ожидании — по статусам задач), % по часам, факт-часы,
  токены, ≈$ (затраты — только Owner/Admin, как на обзоре).
- Таблица задач: ID, название, тип, статус, план-часы, факт-часы, токены, ≈$, исполнитель.
- «Модели AI»: строка = модель; время/токены/кэш/≈$; `unknown` — затраты, не атрибуцированные
  моделям (дни без подходящих сессий); `no-agent` — время без AI-сессий. Итоги сходятся
  с уровнем задачи (выверено в эпохе 3).
- «Нераспределённое» (затраты без привязки к задаче) — НЕ на этой странице: по данным
  timechecker оно не привязывается к проекту (зафиксировано в спеке 3.2);
  покажем суммой на обзоре портфеля отдельной строкой под Гантом (войдёт в t27).
- Развёрнутость по умолчанию: текущая эпоха и текущий спринт раскрыты, остальное свёрнуто.
- Employee: только свои проекты (membership, t28) и без колонок затрат; Client — нет доступа.
