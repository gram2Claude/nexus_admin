# Спека эпохи 7 — страница «Профиль» (NEXADM-34,35)

Статус: **ОБСУЖДЕНИЕ** — схема на согласование.

## 1. ASCII-схема страницы /profile — НА СОГЛАСОВАНИЕ

Доступ: все аутентифицированные роли (каждый управляет только своей учёткой).
Вход: дропдаун в шапке (аватар «Олег / Owner») → новый пункт **«Профиль»** (над «Тёмная тема»).

```
┌─ Профиль ────────────────────────────────────────────────┐
│                                                          │
│  ┌─ Данные аккаунта ─────────────────────────────────┐   │
│  │ Email   gram2.0ai@gmail.com        (только чтение)│   │
│  │ Роль    ● Owner                    (только чтение)│   │
│  │ Имя     [ Олег                   ]  [Сохранить]   │   │
│  │   └ отображается в шапке и в колонке «Исполнитель»│   │
│  └───────────────────────────────────────────────────┘   │
│                                                          │
│  ┌─ Смена пароля ────────────────────────────────────┐   │
│  │ Текущий пароль   [ ••••••••••••     ]             │   │
│  │ Новый пароль     [ ••••••••••••     ]  мин. 12    │   │
│  │ Ещё раз          [ ••••••••••••     ]             │   │
│  │                            [Сменить пароль]       │   │
│  │ ✓ Пароль изменён  /  ✗ Текущий пароль неверен     │   │
│  └───────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────┘
```

Правила:
- **Смена пароля**: обязательная проверка текущего пароля (bcrypt-compare); новый —
  минимум 12 символов, два поля должны совпадать; rate-limit как на логине
  (5 попыток / 15 мин — защита от перебора текущего пароля изнутри сессии);
  после смены текущая сессия остаётся живой (JWT), остальные устройства
  отвалятся при следующей ревалидации роли (до 10 мин) — норм для нашего масштаба.
- **Имя**: trim, 1–60 символов; пустое → ошибка. Email и роль из профиля не меняются
  (роль — только через раздел «Пользователи» Owner/Admin'ом; email — идентификатор входа).
- Серверные actions с единым контрактом `{error}`; никакого логирования паролей.
- Обе темы, как обычно.
