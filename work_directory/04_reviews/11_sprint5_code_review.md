## Принято как ограничение (документировано)

- Карточки обзора — div+router.push (нет middle-click/клавиатуры); строки задач недоступны
  с клавиатуры — UX-полировка после первой очереди.
- Факт проекта = ВСЕ строки v_task_fact слага (включая задачи вне дерева кабинета) — самое
  честное «итого потрачено»; эпохи/спринты могут суммироваться чуть меньше (задокументировано в коде).
- ORDER BY ext_id лексикографический — корректно до 99 задач на проект (зеро-пэддинг канона).
