## Принято как ограничение (документировано)

- Тултипы эпох недоступны по тапу (планшет) — Radix Tooltip hover-only; планшетный сценарий
  вне скоупа первой очереди, вернёмся при необходимости (Popover по клику).
- Год в подписях месяцев не выводится — диапазон портфеля сейчас < 3 месяцев.
- Кэша на roll-up запросах нет — на текущем объёме (148 задач) накладные копеечные.

## Чисто (проверено обоими)

(Portal, уникальные ключи); next-themes без FOUC (блокирующий скрипт, suppressHydrationWarning,
тоггл внутри lazy-монтируемого меню); server action описаний (RBAC+UUID+rowCount); SQL
